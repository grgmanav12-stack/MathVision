# 🌾 MathVision – Nagaland Agriculture Analytics

A Streamlit web app for visualizing and forecasting agricultural production data for Nagaland, India.

## Features
- Secure login / signup system
- District-wise crop production analysis
- Linear & polynomial regression with 3-year forecasts
- Interactive charts powered by Plotly
- Live demo with custom data input

## Run Locally
```bash
pip install -r requirements.txt
streamlit run mathvision_app_v4.py
```
